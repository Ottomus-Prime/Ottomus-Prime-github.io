Viewer Commands

COMMAND USAGE DESCRIPTION
!discord !discord Displays a link to my Discord server
!love !love <target> Show the target viewer some love!
!hug !hug <target> Give the target viewer a hug!
!lurk !lurk Announce the desire to lurk
!quote !quote || !quote <number> Display a random quote || display a quote by its number
!heist !heist <amount> Start/join a heist to get points!
!gamble !gamble <amount> Gamble to get points!
!give !give <target> <amount> Give points to the target viewer
!points !points Display how many points you have
!top !top Display the top viewers based on points
!tophours !tophours Display the top viewers based on hours watched
!8ball !8ball <yes/no question> Ask the 8ball a question and receive a non-commital answer
!slots !slots <amount> Try your luck at the slots for more points!
!duel !duel <target> <amount> Request to duel another viewer for points
!accept !accept Accept a duel challenge from another viewer
!deny !deny Deny a duel challenge form another viewer
!cancel !cancel Cancel a duel you have initiated
!death !death Display current number of deaths on the death counter
          

Mod Commands

COMMAND USAGE DESCRIPTION
!addcommand !addcommand !<name> <content> Add a command directly from chat
!editcommand !editcommand !<name> <new content> Edit a command directly from chat
!removecommand !removecommand !<name> Remove a command directly from chat
!enable !enable !<name> Enable/disable a command directly from chat
!addquote !addquote <quote> Add a quote through chat
!quoteedit || !editquote !quoteedit || !editquote <number> <quote> Edit a quote through chat
!quoteremove || !removequote !quoteremove || !removequote <number> Remove a quote through chat
!addpoints !addpoints <target> <amount> Add points to a viewer
!removepoints !removepoints <target> <amount> Remove points from a viewer
!settitle !settitle <title> Changes the stream title
!setgame !setgame <game> Changes the stream game
!death + !death + Increases the death counter by 1
!death - !death - Decreases the death counter by 1
!death !death <number> Sets the death count to a specific number
